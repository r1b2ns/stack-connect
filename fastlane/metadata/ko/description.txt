StackConnect는 앱 개발자를 위한 올인원 모바일 동반자입니다. 하나의 아름다운 네이티브 iOS 앱에서 App Store Connect와 Firebase 계정을 관리하세요 — 더 많은 연동 기능이 곧 추가됩니다.

주요 기능

App Store Connect
• 검색, 정렬, 즐겨찾기로 모든 앱 탐색
• 버전 메타데이터 보기 및 편집: 설명, 새로운 기능, 프로모션 텍스트, 키워드, URL
• 모든 플랫폼(iOS, macOS, tvOS, visionOS)의 앱 버전 관리
• 심사 제출, 심사 취소, 출시, 버전 거부
• 앱 정보 편집: 번들 ID, SKU, 카테고리, 콘텐츠 권리, 연령 등급
• 로케일별 앱 개인정보 보호 선언 관리
• 기기 유형별 접근성 선언 관리
• 무한 스크롤로 고객 리뷰 탐색, 평점별 필터
• 앱에서 직접 리뷰에 답변하고 응답 삭제
• 일별 차트로 앱 분석 보기: 설치, 삭제, 다운로드, 노출, 페이지 조회
• TestFlight 베타 그룹 및 테스터 관리
• 베타 그룹에 빌드 추가, 팀원 초대
• 팀원 및 역할 관리
• 앱 기록 및 심사 제출 추적

Firebase
• Firebase 프로젝트 및 앱 탐색
• Remote Config 매개변수 및 조건 관리
• 분석 대시보드 보기
• 대상 타겟팅으로 Cloud Messaging 캠페인 전송

다중 계정 관리
• 제공자별 여러 계정 추가
• 계정별 세분화된 권한 (앱, 버전, 사용자, 리뷰, TestFlight, 분석)
• 권한 계층 구조: 보기, 편집, 삭제, 추가
• 오프라인 우선: 로컬 캐시에서 즉시 로드, 백그라운드에서 API 동기화
• 연쇄 삭제: 계정 제거 시 모든 관련 데이터 정리

보안 계정 공유 (Team 및 Lifetime 플랜)
• 암호화된 .scexport 파일로 계정 내보내기
• 비밀번호 보호를 통한 AES-256-GCM 암호화
• 팀원과 액세스 공유
• 수신자가 비밀번호로 파일을 가져와 즉시 액세스
• 내보내기 시 리소스별 권한 선택

추가 연동 기능 출시 예정
• StackConnect는 지속적으로 성장하고 있습니다 — App Store Connect와 Firebase 외에도 더 많은 개발자 플랫폼 연동이 준비 중입니다. 기대해 주세요.

개인정보 보호 최우선: 모든 자격 증명은 iOS 키체인에 안전하게 저장됩니다. 계정 데이터는 SwiftData로 로컬에 캐시됩니다. 분석 없음, 추적 없음, 제3자 서버 없음 — 오직 당신과 Apple/Google의 공식 API만 사용합니다.
