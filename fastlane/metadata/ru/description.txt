StackConnect — это универсальный мобильный помощник для разработчиков приложений. Управляйте аккаунтами App Store Connect и Firebase из единого, красивого нативного приложения для iOS — скоро появятся новые интеграции.

ОСНОВНЫЕ ВОЗМОЖНОСТИ

App Store Connect
• Просматривайте все приложения с поиском, сортировкой и избранным
• Просматривайте и редактируйте метаданные версий: описание, что нового, рекламный текст, ключевые слова, URL-адреса
• Управляйте версиями приложений на всех платформах (iOS, macOS, tvOS, visionOS)
• Отправляйте на проверку, отменяйте проверку, публикуйте и отклоняйте версии
• Редактируйте информацию о приложении: bundle ID, SKU, категория, права на контент, возрастной рейтинг
• Управляйте декларациями конфиденциальности приложений по локалям
• Управляйте декларациями доступности по семействам устройств
• Просматривайте отзывы клиентов с бесконечной прокруткой, фильтруйте по рейтингу
• Отвечайте на отзывы и удаляйте ответы прямо из приложения
• Просматривайте аналитику приложения с ежедневными графиками: установки, удаления, загрузки, показы, просмотры страниц
• Управляйте бета-группами и тестировщиками TestFlight
• Добавляйте сборки в бета-группы, приглашайте членов команды
• Управляйте членами команды и ролями
• Отслеживайте историю приложения и отправки на проверку

Firebase
• Просматривайте проекты и приложения Firebase
• Управляйте параметрами и условиями Remote Config
• Просматривайте панель аналитики
• Отправляйте кампании Cloud Messaging с таргетингом аудитории

Управление несколькими аккаунтами
• Добавляйте несколько аккаунтов на провайдера
• Детальные разрешения для каждого аккаунта (приложения, версии, пользователи, отзывы, TestFlight, аналитика)
• Иерархия разрешений: просмотр, редактирование, удаление, добавление
• Офлайн-режим: мгновенная загрузка из локального кеша, синхронизация с API в фоне
• Каскадное удаление: удаление аккаунта очищает все связанные данные

Скоро новые интеграции
• StackConnect активно развивается — помимо App Store Connect и Firebase, на подходе интеграции с другими платформами для разработчиков. Следите за обновлениями.

Конфиденциальность прежде всего: все учётные данные надёжно хранятся в iOS Keychain. Данные аккаунтов кешируются локально с помощью SwiftData. Без аналитики, без отслеживания, без сторонних серверов — только вы и официальные API Apple/Google.
