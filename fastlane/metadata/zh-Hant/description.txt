StackConnect 是應用程式開發者的全方位行動助手。透過一款精美的原生 iOS 應用程式，即可管理您的 App Store Connect 和 Firebase 帳號——更多整合即將推出。

主要功能

App Store Connect
• 瀏覽所有應用程式，支援搜尋、排序和收藏
• 檢視和編輯版本中繼資料：描述、新功能、推廣文字、關鍵字、網址
• 管理所有平台的應用程式版本（iOS、macOS、tvOS、visionOS）
• 提交審查、取消審查、發佈和拒絕版本
• 編輯應用程式資訊：Bundle ID、SKU、類別、內容版權、年齡分級
• 依語言管理應用程式隱私聲明
• 依裝置系列管理無障礙聲明
• 無限捲動瀏覽客戶評論，依評分篩選
• 直接在應用程式中回覆評論和刪除回應
• 透過每日圖表檢視應用程式分析：安裝、刪除、下載、曝光、頁面瀏覽
• 管理 TestFlight 測試群組和測試人員
• 將建置版本新增至測試群組，邀請團隊成員
• 管理團隊成員和角色
• 追蹤應用程式歷史記錄和審查提交

Firebase
• 瀏覽 Firebase 專案和應用程式
• 管理 Remote Config 參數和條件
• 檢視分析儀表板
• 發送 Cloud Messaging 活動，支援受眾定向

多帳號管理
• 每個供應商可新增多個帳號
• 每個帳號的精細權限控制（應用程式、版本、使用者、評論、TestFlight、分析）
• 權限層級：檢視、編輯、刪除、新增
• 離線優先：從本機快取即時載入，在背景與 API 同步
• 級聯刪除：移除帳號會同時清除所有相關資料

安全帳號共享（團隊版和終身版方案）
• 將帳號匯出為加密的 .scexport 檔案
• AES-256-GCM 加密，密碼保護
• 與團隊成員共享存取權限
• 收件人使用密碼匯入檔案，即可立即獲得存取權限
• 匯出時可針對每項資源選擇權限

更多整合即將推出
• StackConnect 正在積極成長——除了 App Store Connect 和 Firebase 之外，更多開發者平台整合即將到來。敬請期待。

隱私至上：所有憑證安全儲存在 iOS 鑰匙圈中。帳號資料透過 SwiftData 在本機快取。沒有分析、沒有追蹤、沒有第三方伺服器——只有您和 Apple/Google 的官方 API。
